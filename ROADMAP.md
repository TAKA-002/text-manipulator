# Roadmap

Text Manipulator の今後の開発予定をまとめたファイルです。
優先度は未定。少しずつ実装・テストを行いながら進めていきます。

---

## 新機能

### 変換系
- [ ] 大文字/小文字変換（uppercase / lowercase / titlecase）
- [ ] 改行コード変換（LF ↔ CRLF）

### エンコード系
- [ ] URL エンコード / デコード
- [ ] Base64 エンコード / デコード
- [ ] HTML エスケープ / アンエスケープ

### 整形系
- [ ] 前後の空白トリム
- [ ] JSON フォーマット / 圧縮
- [ ] 重複行の削除
- [ ] 行ソート（昇順 / 降順）

### テキスト解析
- [ ] 文字数・行数カウンター（リアルタイム表示）

### 置換強化
- [ ] 正規表現置換
- [ ] 複数の置換ルールを一括適用

### 履歴
- [ ] 変換履歴の保持・再呼び出し

---

## UX 改善

- [ ] 差分ハイライト表示（入力と出力の変更箇所を背景色で強調）
- [ ] ドラッグ＆ドロップでファイル読み込み
- [ ] キーボードショートカット一覧のヘルプ表示
- [ ] ダークモード
